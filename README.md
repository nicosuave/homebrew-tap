# Homebrew Tap

Personal Homebrew tap for CLI tools.

## Installation

```bash
brew tap nicosuave/tap
```

## Formulas

### memex

Fast local history search for Claude and Codex logs.

```bash
brew install nicosuave/tap/memex
```

### agentexport

Share Claude Code and Codex session transcripts with end-to-end encryption.

```bash
brew install nicosuave/tap/agentexport
```
